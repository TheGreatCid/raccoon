CrystalPlasticitySlipRateGSS

Description

CrystalPlasticitySlipRateGSS derives from CrystalPlasticitySlipRate which calculates slip rate in the crystal plasticity system. The slip system is read into this class. The Schmid tensor is generated in this class and the flow direction is calculated by calcFlowDirection(). The slip rate is a function of PK2 stress T, slip resistance s and state variables .
